Perguntas e Respostas

O que é Button Box

Button Box é um dispositivo elaborado para adcionar funções a um determinado simulador, por exemplo, você conecta o dispositivo em seu computador e indica que o botão número 1 será o botão de ignição.

Posso utilizar qualquer Button Box para qualquer simulador?

Sim. Na realidade todos os dispositivos serão reconhecidos pelo computador entretanto, sugerimos utilizar um dispositivo que mais se adeque ao seu simulador pois a idéia é ter o máximo de imersão, sendo assim um dispostivo pode se tornar mais confortavel que outros.  

Eu utilizo varios simuladores, qual dispositivo devo escolher?

Se você utiliza varios simuladores, sugerimos um dispositivo com maior amplitude de comandos, algo que se adeque em varias plataformas. Desta maneira, você otimiza seu cockpit.

Preciso instalar algum driver?

Não. Todos os dispositivos são Plug and Play, ou seja, basta conectar a uma porta USB  e será reconhecido como um dispositivo HID pelo computador.

Quais configurações preciso fazer no dispositivo?

Na realidade a configuração é no simulador onde você indica qual botão fará determinada função.

Posso personalizar um dispositivo?

Sim. Muitos clientes entram em contato com uma idéia, e juntos vamos lapidando até chegar no dispositivo ideal. Portanto se você tem alguma idéia, fale conosco para juntos torná-la realidade.
